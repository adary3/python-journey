# Python Basics Day 0 - Kleeon Empire

Small Flask demo used for learning.

Run locally:

```cmd
python -m pip install -r requirements.txt
python app.py
```

Open in your browser (Chrome): `http://127.0.0.1:5000/`

Features:
- Visitor counter stored in `visitors.txt`
- Todo list loaded from `my_missions.txt` and a form to add items
- Dark mode toggle that persists via a cookie
- Guestbook: `day6-guestbook.py` stores messages in `guestbook.json` and shows them on `/`

Notes:
- `my_missions.txt`, `visitors.txt`, and `guestbook.json` are local files created/updated by the app.
